# OatmealRazing Plugins

Plugin marketplace for Claude Code and other AI coding agents.

## Install

```bash
claude plugin marketplace add OatmealRazing/oatmealrazing-plugins
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `dev-context-methodology` | Project lifecycle tracking and context management using DCM |
| `daythread` | Personal life task tracking and reminders for Claude Code |

### Install a plugin

```bash
claude plugin install dev-context-methodology
```
